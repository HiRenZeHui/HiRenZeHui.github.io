快速更改博客上的主题

上线这个博客，hexo的默认的模板比较丑，于是自己就找了其他hexo的模板，于是在网上参考了如何更换
博客的方法。大部分都是使用命令行的方式。

首先使用打开你的安装hexo的目录

在此目录下打开git,输入命令

网上大部分都是这种方法，发现这种方式下载主题太慢了，于是直接下载喜欢的主题，解压后放在themes文件夹下

然后修改_config.yml文件，把里面的themes: landscape更换为themes: yilia (yilia是下载的解压后主题文件名),接着再打开
git 部署上传到github上面就大功告成了。
